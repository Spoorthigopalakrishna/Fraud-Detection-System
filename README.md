# Fraud Detection System

This repository contains the code for a full-stack Fraud Detection System.

## Structure
- `backend/`: Flask API and model training scripts.
- `frontend/`: React-based user interface dashboard, which also houses `data/` and `reports/`.
- `notebooks/`: Exploratory Data Analysis (EDA) notebooks.
